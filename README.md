# GiveMeHeals

A World of Warcraft click-cast healing addon for Restoration Shamans.

**Created by Xerra - Spinebreaker EU**

## Overview

GiveMeHeals lets you cast healing spells directly from unit frames using mouse clicks and modifier keys. Simply hover over a party or raid frame and click to heal -- no targeting required.

## Spell Bindings

| Input | Spell |
|---|---|
| Left Click | Riptide |
| Shift + Left Click | Healing Wave |
| Ctrl + Left Click | Healing Surge |
| Alt + Left Click | Chain Heal |
| Right Click | Purify Spirit |
| Shift + Right Click | Earth Shield |
| Ctrl + Right Click | Unleash Life |
| Middle Click | Healing Rain |

## Supported Frames

- Raid frames (Blizzard Compact Unit Frames)
- Party frames
- Player frame
- Target frame
- Focus frame

## Features

- **Click-cast healing** on all standard unit frames
- **Healing tooltip** showing all available spell bindings on hover (toggle on/off)
- **Movable game tooltip** -- reposition the default tooltip anywhere on screen
- **Minimap button** for quick access to settings (right-click drag to reposition)
- **Class and spec validation** -- warns if you're not a Restoration Shaman
- **Persistent settings** -- all positions and preferences are saved between sessions

## Usage

1. Left-click the minimap button to open the settings panel.
2. Toggle the healing tooltip on or off to show spell bindings when hovering over unit frames.
3. Use the "Move General Tooltip" button to reposition the game tooltip.
4. Hover over any supported unit frame and use the mouse/modifier combinations above to cast spells.

## Installation

1. Download and extract the `GiveMeHeals` folder into your WoW addons directory:
   ```
   World of Warcraft/_retail_/Interface/AddOns/
   ```
2. Restart WoW or type `/reload` in chat.
3. Ensure GiveMeHeals is enabled in the AddOns menu at the character select screen.

* This addon is hosted at CurseForge and can be downloaded from there if manual install not an option.
